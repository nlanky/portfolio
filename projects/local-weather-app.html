Local Weather App - HTML5, CSS3, Sass, BS4, jQuery

This application provides the local weather forecast to the user based on their location. The layout was constructed with HTML5, CSS3, Sass and Bootstrap (version 4). The functionality is provided by jQuery and an API. Once the user allows the browser access to their location, the app will generate the temperature and summary of the weather at their location. The user can also switch between Celsius and Fahrenheit with one click.
