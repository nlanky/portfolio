Hartley Photography - WordPress

This site was created using HTML5, CSS3, the Bootstrap front-end responsive framework (version 4), Sass and the jQuery JavaScript library. It also makes use of the Font Awesome font and icon toolkit for the footer icons. On this site I have demonstrated an understanding of the newest version of Bootstrap and the changes made to some CSS classes (in the navigation bar, img-responsive->img-fluid etc.) as well as a grasp of jQuery functions to add more functionality. jQuery is used to change the active link as the user scrolls down, toggle the responsive menu on small devices and add a neat effect on the form labels when they are filled. I have also taken advantage of Sass which makes it easier to interpret the stylesheet and allows for easy testing of different colour palettes.
